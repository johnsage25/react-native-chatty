# Installation

Run `npm install react-native-chatty`


:::tip Using Yarn

If you're installing via Yarn make sure to install [peer dependencies](#peer-dependencies). Most likely you have them already installed if you're using React Navigation.
:::



## Peer Dependencies
-    `dayjs`
-    `react-native-gesture-handler`
-    `react-native-reanimated`
-    `react-native-safe-area-context`
-    `recyclerlistview`


## Optional Dependencies

:::tip Install with caution.

Those packages are totally optional, you will need install it *only* when you want to use specific feature.

:::

- `react-native-parsed-text`
- `react-native-image-viewing`
- `react-native-context-menu-view`
- `lottie-ios && lottie-react-native`
- `moti`
- `expo-linear-gradient`
- `expo-image-picker` OR `react-native-image-picker`
- `expo-haptics` OR `react-native-haptic-feedback`
